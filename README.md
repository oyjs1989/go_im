# go_im
Instant Messaging with Golang

# 核心技术
 gin GORM swagger logrus auth

 # 核心功能
- 用户模块
    - 基础信息
    - 好友
    - 群
- 消息
    - 推送/接收
    - 内容 图片/文件/视频
    - 单聊 群聊 
    - mongodb 历史消息
    - mysql 历史备份
    - redis 缓存
    - 推送

- 发送和接受消息，文字，文件，访客，广播，快速回复，撤回，心跳检测

# 技术栈
- websocket
- channel/goroutine
- gin
- template
- gorm
- sql
- nosql
- mq
- rpc
